# Shop

Веб-приложение интернет-магазина на FastAPI с каталогом товаров, корзиной, заказами и рекомендательной системой.

## Что реализовано

- Каталог товаров и категории
- Поиск по названию и описанию
- Корзина для авторизованных и гостей (через `session_id`)
- Оформление заказа
- Блоки рекомендаций:
  - Популярное
  - Рекомендуем сейчас
  - Дополни заказ (в корзине)
- События взаимодействий для ML-датасета

## Стек

- Python 3.11+
- FastAPI
- SQLAlchemy
- Jinja2
- SQLite
- Uvicorn
- (опционально) scikit-learn, pandas, numpy, ctgan

## Быстрый старт

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить приложение:

```bash
uvicorn main:app --reload
```

3. Открыть в браузере:

- Главная: `http://127.0.0.1:8000/`
- Swagger: `http://127.0.0.1:8000/docs`

## Структура проекта

- `main.py` — инициализация приложения и HTML-страницы
- `routers/` — API роуты (users, products, cart, orders, recommendations)
- `templates/` — Jinja2-шаблоны
- `static/` — CSS/JS/изображения
- `docs/` — документация по MVP рекомендаций и генерации данных
- `scripts/` — сервисные и тестовые скрипты

## Рекомендации и ML-данные

- Основная логика рекомендаций: `recommendations.py`
- API рекомендаций: `routers/recommendations.py`
- ML-датасет взаимодействий: `ml_dataset.py`, `routers/ml_dataset.py`

## Примечания

- Файл базы данных и служебные артефакты исключены из git через `.gitignore`.
- Для гостей используется `session_id`, который фронтенд передает в запросах корзины.
